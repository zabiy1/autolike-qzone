# 空间妹自动点赞助手 (Mac/Win) 使用手册

> **项目说明**：本工具 Fork 自 GitHub 仓库 [auto-liking-qzone-windows](https://github.com/help660vip/auto-liking-qzone-windows)，针对 macOS 环境及稳定性进行了适配与重构。

---

### 🚀 使用步骤

1. **启动程序**
   双击运行 `空间妹自动点赞助手.exe` (Mac用户请运行打包好的可执行文件)。
   
2. **账号输入**
   当命令行窗口弹出 `请输入你的QQ号：` 提示后，输入你需要点赞的 QQ 号码并按下 **回车键**。

3. **安全登录**
   程序会自动调起系统浏览器。请在弹出的窗口中通过 **扫码** 或 **账号密码** 正常登录 QQ 空间。
   > **注意：** 登录成功后请 **不要手动关闭浏览器**，也不要进行任何多余操作。程序在检测到登录信号后会自动关闭浏览器并保存配置。

4. **挂机巡检**
   浏览器关闭后，请保持主窗口开启。程序会进行初始化，稍等一分钟左右，控制台显示 `[巡检] 发现新动态` 时，即代表自动点赞功能已正式开启。

---

### 💡 多账号支持方案

如果你需要同时挂载多个 QQ 账号，请执行以下操作：
* 在电脑中创建不同的文件夹（例如：`账号1`、`账号2`）。
* 将 `空间妹自动点赞助手.exe` 分别复制到这些文件夹中。
* 在每个文件夹下独立运行即可实现多开，配置互不干扰。
